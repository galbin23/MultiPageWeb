# My Professional Multi-Page Website

## Site Description
This is a small, professional website built with semantic HTML, accessibility features, responsive media, consistent navigation/footer, and a contact form. It is currently very basic as there is no CSS involved. CSS will be added for the next assignment.

## List of Pages
- Home (index.html)
- About (about.html)
- Portfolio (portfolio.html)
- Contact (contact.html)
