# My Professional Multi-Page Website

## Site Description
This is a small, professional website built with semantic HTML, accessibility features, responsive media, consistent navigation/footer, and a contact form. It demonstrates best practices for web development.

## List of Pages
- Home (index.html): Introduction with hero image.
- About (about.html): Background and skills.
- Portfolio (portfolio.html): Projects with image and YouTube embed.
- Contact (contact.html): Form for inquiries.
